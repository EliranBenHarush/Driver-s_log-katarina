🔥 חיבור ל-Firebase

שלבים לחיבור חינמי:
1. לך ל-console.firebase.google.com
2. צור פרוייקט חדש → הוסף אפליקציה Web (</>)
3. העתק את אובייקט firebaseConfig
4. הדבק אותו כאן (JSON בלבד, ללא const firebaseConfig =)
⚠️ חשוב: בממשק Firebase → Firestore Database → Rules, שנה ל:
allow read, write: if true; (לשימוש פרטי בלבד)
